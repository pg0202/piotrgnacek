Snapstop

Snapstop is a first person puzzle game where you solve puzzles with the help of a camera that can pause objects in time. Pause doors, buttons and physics objects in order to solve puzzles.

Learning Outcomes

  • Team Size

  • Software Used
    Unreal Engine 5.4 (Blueprints + C++), Blender

My Contributions

  • Created and scripted puzzle levels, balancing difficulty to encourage creative problem-solving.
  • Built interactive systems (door triggers, pressure plates, object physics) to work dynamically with the camera mechanic.
  • Designed player interaction systems and mapped intuitive controls.
  • Playtested and iterated on puzzle design to improve clarity, flow, and player engagement.
